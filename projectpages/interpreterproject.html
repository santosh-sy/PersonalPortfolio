Core Language Interpreter

A Java based Interpreter

In my interpreter project, I designed and implemented a language interpreter for the "Core" language, utilizing Java. I formulated finite state automata and implemented the tokenizer, parser, and executor.

I created a robust interpreter capable of handling complex control flow, variable assignments, mathematical operations, and function calls. Through meticulous testing and debugging, I ensured accurate execution of "Core" language programs.

This project enhanced my skills in language design, compiler theory, and implementation. I tackled challenges in data types, memory allocation, and performance optimization, showcasing my ability to comprehend programming languages and create reliable systems.
