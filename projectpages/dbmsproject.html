Building a User-Friendly Database

In my database creation project, I collaborated to construct a relational database for a library, documenting books, films, and songs. We designed an efficient database schema, establishing relationships between entities for easy access and management.

Using Java and SQL, I developed the user interface, enabling customers to search, check availability, and manage borrowing records. The intuitive interface enhanced the user experience.

Throughout the project, I gained valuable experience in database design, SQL programming, and data modeling. This project showcased my ability to design robust databases and deliver practical solutions for effective data management.
