Teachers' LoungeApplication

Mobile based iOS Application

In my capstone project at Ohio State University, I developed a teacher social media application using React Native, SQL, AWS, and NodeJS. The goal was to create a collaborative platform for K-12 educators, allowing them to exchange resources, teaching supplements, and ideas seamlessly across multiple devices.

To ensure scalability and security, I leveraged AWS services and deployed a robust backend infrastructure using NodeJS. This integration facilitated seamless communication between the application and the server, enabling teachers to connect and collaborate effectively.

Throughout the project, I demonstrated my proficiency in React Native for mobile app development, SQL for database management, AWS for scalable infrastructure, and NodeJS for backend integration. This capstone project showcased my ability to utilize these skills to create practical solutions that enhance collaboration and support within the education community.
